Colophon / 技术与偏好

这一页是给好奇的人看的：这座「借宿地球」观测站，是怎么被搭起来的，以及它打算如何在宇宙尘埃里长时间运行。

技术栈

  • 生成器：Hexo 8.x
  • 主题：自定义主题 lodging（深空背景 + 粒子拖尾 + 慢速评论 + Ambient Mode）
  • 部署：GitHub Pages · 仓库 borrowed-orbit.github.io

我希望这个站点尽量保持：

  • 依赖简单，可长时间维护；
  • 不依赖数据库，所有内容都是可以被 git 版本控制的纯文本；
  • 即使很久不更新，几年后 git clone + 一条命令也能重新跑起来。

写作偏好

  • 介质：Markdown 为主，尽量少图，多文字；
  • 节奏：不追热点，也不会为流量刻意调整选题；
  • 时间：大部分文字会在深夜或清晨完成，此时世界比较安静；
  • 语言：以中文为主，偶尔会引用英文术语和原始表述。

站点小规则（暂定）

  1. 不卖广告
    这里不会出现闪烁的横幅、浮动的推广位，也不会有隐蔽的软广。

  2. 不做 SEO 体操
    文章标题会尽量贴近内容，而不是贴近搜索引擎。

  3. 旧文不删，只加后记
    如果将来的我不同意现在的自己，我会在文末写一段后记，而不是把它抹去。

  4. 不追更新频率 KPI
    有话要说的时候才写，没有的时候就安静地生活。

  5. 尊重慢速交流
    评论区是「Slow Comments」，还有一个单独的「发送信号」页面。这里欢迎认真、缓慢的反馈，不欢迎吵架。

如果有一天你发现这个页面更新了很多次，那大概说明：

我依然在维护这座小小的「深空建筑」，并且还在学习如何更好地栖居于此。
